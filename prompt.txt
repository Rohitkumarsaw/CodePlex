Build a fully responsive premium digital product selling website using only HTML, CSS, and Vanilla JavaScript.

Important:
- The entire project must be created in exactly 4 files only:
  1. index.html
  2. style.css
  3. script.js
  4. data.js
- Do not use any framework.
- Do not use any backend.
- Do not use any build tools.
- Do not create extra files.
- All product data must be managed from data.js.
- The website must look premium, modern, handcrafted, smooth, and production-quality, not like a generic AI template.
- The theme must support both dark neon mode and elegant light mode with smooth transitions.
- The entire website must be 100% responsive for mobile, tablet, laptop, and desktop.

==================================================
PROJECT GOAL
==================================================

I want a digital product selling website where I can:
- add products
- edit products
- delete products
- manage all products from data.js driven architecture
- show products category-wise
- search and filter products
- support both free and paid digital products
- show detailed product popup/modal on click
- show multiple download links
- show payment info for paid products
- verify serial key for paid products before showing downloads
- show direct downloads for free products
- use a premium responsive UI with a dark neon style and smooth light mode

This project is frontend-only, so structure the code in a scalable way. If any feature needs real backend security in production, implement the best possible frontend demo architecture and clearly separate the logic so it can be upgraded later.
